# UI Design Project 🎨

A simple and responsive **UI Design web page** built using **React.js** and styled with **CSS/Tailwind**.  
This project showcases a clean layout, reusable components, and modern design practices.

---

## 🚀 Features
- Modular React component structure
- Clean design with cards and sections
- Sticky header for easy navigation
- Smooth scroll behavior with hidden scrollbars
- Background image integration and Lucide icons

---

## 🛠️ Tech Stack
- **React.js** – Component-based UI
- **CSS / Tailwind CSS** – Styling and responsiveness
- **Lucide Icons** – Modern icon set
